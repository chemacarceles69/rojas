WEB CARMEN ROJAS

Estructura incluida:
- index.html
- cronologia.html
- hemeroteca.html
- galeria.html
- css/styles.css
- js/data.js
- js/main.js
- imágenes organizadas en /img

Para abrir la web:
1. Descomprime el ZIP.
2. Abre index.html en tu navegador.

Nota:
- La portada actual usa img/portada.jpg.
- Si quieres cambiar la foto de portada, sustituye ese archivo por la imagen definitiva manteniendo el mismo nombre.
- Los datos de hemeroteca y galería están en js/data.js para poder ampliar la web fácilmente.
